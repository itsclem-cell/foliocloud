Built for everyone

Accessibility is in every detail.

This site is built to the same standard we deliver for clients. Every feature listed here is active and testable right now.

⌨️

Full keyboard navigation

Every interaction reachable without a mouse. Logical tab order maintained throughout.

🔊

Screen reader optimised

Semantic HTML5 landmarks, ARIA labels, live regions, and meaningful alt text throughout.

🎨

Contrast modes

High contrast (black/white), yellow-on-black, and dark mode — all switchable instantly from the toolbar above.

📖

Typography controls

Increase text size, switch to a plainer font, or use a dyslexia-friendly typeface from the toolbar above.

💫

Reduce motion

All animations and transitions can be disabled for people with vestibular disorders or motion sensitivity.

📋

Accessible forms

Clear labels, error messages, inline validation, and ARIA live regions on every form across the site.

📱

Responsive & mobile-accessible

Fully functional on all screen sizes. Touch targets meet WCAG 2.2 minimum size guidelines.

🖨️

Print-ready

Styled print stylesheet removes decorative elements and expands links for printed output.
